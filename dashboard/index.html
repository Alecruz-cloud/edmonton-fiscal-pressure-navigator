Edmonton Fiscal Pressure Navigator

Identifies at most three financial pressure areas that deserve deeper management review before a budget or capital-planning cycle. It supports prioritization for review — it does not make policy decisions or assess financial health.

Review priorities

No more than three signals, ranked under governance › pressure_rules.md · v1.2. A signal is a transparent flag for human review — never a score. High review priority requires at least two independent tests, including one Edmonton historical test. Calgary and Alberta-city evidence is supporting context only.

Verdict board — every metric evaluated this run rules v1.2 · full evidence published

All evaluated metrics with their verdicts, not only the surfaced signals — flagged or not, each row opens its full test evidence. This is a governance disclosure, not a KPI scoreboard.

Evidence explorer

The full evaluated set at a glance — every card opens to its Edmonton trend, formula, source trace, data-quality notes, and clearly labelled supporting context.

Alberta context map

Where Edmonton sits among Alberta's reporting cities on the four comparison classes approved in governance › pressure_rules.md · v1.2. Supporting context only — cross-sectional position can never create a high-priority signal, and this view generates none. It is a descriptive display with full coverage disclosure, not a performance ranking.

Bubble position marks the city's census-subdivision centroid; bubble colour encodes the metric value (equal size — city scale is deliberately not encoded). Edmonton is outlined in ink. Zoom is limited to province↔region scale so centroid bubbles are never mistaken for street-level locations; the base map is orientation only. If tiles cannot load (offline), the StatCan outline and bubbles still render.
